# API Reference

FuzeAgent provides a comprehensive REST API for managing organizations, teams, agents, and tasks. All endpoints return JSON responses and use standard HTTP status codes.

## Base URL

```
http://localhost:8006
```

## Authentication

Currently, FuzeAgent operates without authentication for development. Production deployments should implement JWT-based authentication.

## Organizations

### List Organizations

```http
GET /organizations
```

**Response:**
```json
[
  {
    "id": "org_123",
    "name": "ACME Corporation",
    "description": "Main development organization",
    "created_at": "2024-01-15T10:30:00Z"
  }
]
```

### Create Organization

```http
POST /organizations
Content-Type: application/json

{
  "name": "New Organization",
  "description": "Organization description"
}
```

**Response:**
```json
{
  "id": "org_124",
  "name": "New Organization",
  "description": "Organization description",
  "created_at": "2024-01-15T11:00:00Z"
}
```

## WebSocket Events

FuzeAgent provides real-time updates via WebSocket connection:

```javascript
const ws = new WebSocket('ws://localhost:8006/ws');

ws.onmessage = (event) => {
  const message = JSON.parse(event.data);
  
  switch (message.type) {
    case 'organization_created':
      console.log('New organization:', message.data);
      break;
    case 'agent_created':
      console.log('New agent:', message.data);
      break;
    case 'task_updated':
      console.log('Task update:', message.data);
      break;
  }
};
```

## Error Handling

All endpoints return appropriate HTTP status codes:

- `200` - Success
- `201` - Created
- `400` - Bad Request
- `404` - Not Found
- `500` - Internal Server Error

Error responses include details:

```json
{
  "error": "Invalid request",
  "details": "team_id is required",
  "code": "VALIDATION_ERROR"
}
```

## Rate Limiting

API requests are currently unlimited for development. Production deployments should implement appropriate rate limiting.

## SDK Integration

For programmatic access, consider using our SDKs:

```javascript
// JavaScript/TypeScript
import { FuzeAgentClient } from '@fuzeagent/client';

const client = new FuzeAgentClient({
  baseUrl: 'http://localhost:8006'
});

const agents = await client.agents.list({ teamId: 'team_456' });
```

```python
# Python
from fuzeagent import FuzeAgentClient

client = FuzeAgentClient(base_url='http://localhost:8006')
agents = client.agents.list(team_id='team_456')
```
